¿Que son iteradores java?
Iterator es un patrón de diseño de comportamiento que permite el recorrido 
secuencial por una estructura de datos compleja sin exponer sus detalles 
internos. Gracias al patrón Iterator, los clientes pueden recorrer elementos de 
colecciones diferentes de un modo similar, utilizando una única interfaz iteradora.
